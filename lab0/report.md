6. Що таке чисельні методи?
Це алгоритмічні способи наближеного розв’язування математичних задач із використанням обчислювальної техніки. Вони оперують наближеними значеннями та враховують похибки округлення.

Які типи задач розв’язуються чисельними методами?
До основних типів задач належать:

розв’язування систем лінійних рівнянь;

інтерполяція та апроксимація;

чисельне диференціювання та інтегрування;

оптимізація;

розв’язування диференціальних рівнянь.

Які переваги NumPy?
NumPy — це спеціалізована наукова бібліотека Python, яка призначена для ефективної роботи з масивами та розв'язання задач лінійної алгебри. Вона є базовим інструментом для чисельних обчислень завдяки своїй продуктивності.

Для чого використовується Git?
Git — це розподілена система контролю версій, яка дозволяє:

зберігати історію змін коду;

працювати з різними версіями програми;

організовувати як індивідуальну, так і командну роботу.

Чому лабораторні необхідно зберігати у системі контролю версій?
Згідно з вимогами курсу, це необхідно для створення єдиного репозиторію для всіх робіт. Це дозволяє структурувати виконання завдань (кожна робота в окремій папці), фіксувати прогрес за допомогою комітів та мати надійне місце зберігання коду з можливістю його публікації на GitHub.
